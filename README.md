# GameShelf

Gameshelf is a web application that helps you track your video game collection. You can add games, mark them as completed, and organize them by platform. Whether you're planning to play, currently playing, or have completed a game, Gameshelf keeps everything in one place.

Features
Add Games: Add new games with details like name, platform, status, and cover image.

Edit Games: Update game details (name, platform, status).

Delete Games: Remove games from your collection.

Track Progress: Mark games as "Planning to Play," "In Progress," or "Completed."

Dashboard: View statistics like total games, completion rate, and games by platform.

Responsive Design: Works on desktop, tablet, and mobile devices.

Technologies Used
Front End
React: JavaScript library for building the user interface.

React Router: For navigation between pages.

CSS: Styling the app.

RAWG API: Fetching game cover images.

Back End
Node.js: JavaScript runtime for the server.

Express: Web framework for building the API.

MongoDB: NoSQL database for storing game data.

Mongoose: Object Data Modeling (ODM) library for MongoDB.

Tools
Visual Studio Code: Code editor.

Git & GitHub: Version control and hosting.

Postman: Testing API endpoints.
